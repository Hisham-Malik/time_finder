TimeUnitBoundaries class

A class representing the start and end boundaries of a time unit, based on an origin date.

This class is used to encapsulate the start and end DateTime objects that define the boundaries of a specific time unit (e.g., a day, week, month, or year). The boundaries are calculated based on an originDate.

Example:

final boundaries = TimeUnitBoundaries(
  start: DateTime(2024, 9, 1),
  end: DateTime(2024, 10, 1),
  originDate: DateTime(2024, 9, 15),
);

print(boundaries.start); // 2024-09-01 00:00:00.000
print(boundaries.end);   // 2024-10-01 00:00:00.000
print(boundaries.originDate); // 2024-09-15 00:00:00.000
